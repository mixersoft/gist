The research branch is a platform for quick iteration on algorithms and algorithm parameter optimization.

In order to install the prerequisites, you first need to have the .NET 4.0 Framework and Visual C++ 2010 Express or Visual Studio Professional. Open the Visual Studio Command Prompt from the VC++ distribution as an administrator. In this command prompt, navigate to current folder (research\prerequisites), and run the installation script:

install.cmd

After the script finishes, you should be able to go to the research folder in the same command prompt and build the project. Use the command:

msbuild/v:m
